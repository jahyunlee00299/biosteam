# D-Tagatose 생산 공정 (홍조류 바이오매스 기반)

## Process Overview

홍조류 바이오매스를 원료로 하여 D-Tagatose를 생산하는 전체 공정 흐름:

```
홍조류 바이오매스 (세척됨)
    ↓
[1] 황산 가수분해 (Sulfuric Acid Hydrolysis)
    ↓
[2] 중화 및 여과 (Neutralization & Filtration)
    ↓
[2.5] 음이온 수지 교환 (Anion Exchange) - 선택 (시나리오 1)
    ↓ (또는 스킵 - 기본 시나리오)
D-Galactose 함유 용액 (정제 또는 중화)
    ↓
[3] 전세포촉매 전환 (Whole Cell Biocatalysis)
    ↓
D-Tagatose 함유 용액
    ↓
[4] 세포 제거 (Cell Removal)
    ↓
[5] 탈색 (Decolorization)
    ↓
[6] 탈염 (Desalination - 양이온만)
    ↓
[7] 건조 (Drying)
    ↓
D-Tagatose (최종 산물)
```

---

## Detailed Process Steps

### Step 1: 황산 가수분해 (Sulfuric Acid Hydrolysis)

**목적**
- 홍조류 바이오매스의 다당류 구조 분해
- D-Galactose 및 기타 단당류 유리

**조건**
- 반응물: 세척된 홍조류 바이오매스
- 촉매: 황산 (H₂SO₄)
- 반응 온도: 120-140°C
- 반응 시간: 30-60분
- 고액비: 1:5 ~ 1:10

**산출물**
- D-Galactose 함유 산성 용액
- 미반응 고형물 (여과로 제거)

**주요 반응식**
```
Polysaccharides + H₂SO₄ → D-Galactose + D-Mannose + Glucose + ...
```

---

### Step 2: 중화 및 여과 (Neutralization & Filtration)

**목적**
- 과잉 황산 중화
- 불용성 황산염 제거
- pH 조정으로 다음 단계 준비

**중화제 (선택)**
- **옵션 A**: NaOH (수산화나트륨) ★ 추천 (기본 시나리오)
  - 장점: 빠른 반응, 높은 중화 효율, 저비용 ($0.50/kg)
  - 단점: Na⁺ 이온 추가 (후단 탈염에서 양이온 제거 필요)
  - 비용: $0.50/kg × 58 kg (배치) = $29
  - 후단 영향: SO₄²⁻ + Na⁺ → 양이온만 사용 가능

- **옵션 B**: CaOH₂ (수산화칼슘) ★ 대안 (시나리오 1, 소규모)
  - 장점: 저비용 ($0.04/kg), Ca²⁺는 양이온 교환으로만 제거
  - 단점: 반응 속도 느림, 불완전 중화 가능, CaSO₄ 침전 처리 필요
  - 비용: $0.04/kg × 74 kg (배치) = $3 + CaSO₄ 여과/폐기 비용
  - 후단 영향: SO₄²⁻ (음이온) + Ca²⁺ → 음이온 수지 수명 연장

**여과 조건**
- 필터 종류: 0.22 μm 멤브레인 또는 MF (Microfiltration)
- 목적: 미세 고형물, 불용성 황산염 제거

**산출물**
- 중성 D-Galactose 용액 (pH 6.5-7.5)
- 불용성 고형물 (폐기/재활용)

---

### Step 2.5: 음이온 수지 교환 (Anion Exchange) - 선택 (Early Purification)

**목적**
- Step 1 가수분해 중 생성된 **음이온 부산물 제거**
  ├─ SO₄²⁻ (황산염)
  ├─ 레불린산 (Levulinic acid) - 가수분해 부산물 13%
  └─ 포름산 (Formic acid) - 가수분해 부산물 5%
- 반응액 이온 강도 저감 → 촉매 효율 향상
- Step 6 탈염 단계 단순화 (양이온만 필요)

**필요한 경우**
```
시나리오 1 (홍조류 중화 용액) 사용 시 필수:
├─ 음이온 부산물 대량 함유:
│  ├─ SO₄²⁻ (황산)
│  ├─ 레불린산 (13%)
│  ├─ 포름산 (5%)
│  └─ 다른 유기산
├─ 이른 제거로 효율성 향상
├─ 이온 강도 저감 → 전세포촉매 반응 환경 개선
├─ Step 6 단순화 → 양이온만 사용
└─ 결과: 공정 단순화 + 깨끗한 반응액

기본 시나리오 (정제 D-Galactose) 사용 시:
└─ 염분 미량 → 불필요 (skip 가능)
```

**처리 조건**

#### 음이온 수지 선택
- **타입**: Strong Base Anion (SBA)
- **제품**: Amberlite IRA-400, IRA-93, IRA-910
- **형태**: OH⁻ form (수산화물 형태)
- **용량**: 3-5 kg/배치 (1000L 기준)

#### 운전 조건
```
흐름 방향: 상향류 (Upflow) 또는 하향류 (Downflow)
흐름 속도: 2-5 bed volume/시간 (BV/h)
온도: 상온 (20-25°C)
체류시간: 15-30분
처리량: 배치당 100-500L

교환 반응식 (주요 음이온들):
SO₄²⁻ + Anion Resin(-OH)₂ → Anion Resin(-SO₄) + 2OH⁻
HCOO⁻ + Anion Resin(-OH) → Anion Resin(-HCOO) + OH⁻
Levulinic⁻ + Anion Resin(-OH) → Anion Resin(-Levulinic) + OH⁻
위 H⁺ + OH⁻ → H₂O (중성화, 깨끗한 용액)
```

#### 처리 순서 (기본 프로토콜)
```
1. 수지 충전: 컬럼 또는 반응기 하단 → 상향류 흐름
2. 세척: 초순수 2-3 BV로 먼지/공기 제거
3. 활성화: 0.5M NaOH 2-3 BV (OH⁻ form 확인)
4. 처리: D-Galactose 용액 통과 (2-5 BV/h)
5. 검증: 방울 테스트 (phenolphthalein) → 분홍색 = OK
6. 재생: 1M NaOH 3-4 BV (수지 재사용)
   또는 폐기 (소규모일 경우)
```

**산출물**
- 음이온 부산물 제거된 정제 D-Galactose 용액 (pH 6-7)
  ├─ SO₄²⁻ 제거: 99%
  ├─ 레불린산 제거: 95%
  └─ 포름산 제거: 95%
- 포화 음이온 수지 (재생 또는 폐기)
- 용액 순도: 미량 이온만 남음 (Step 6 양이온 제거로 최종 정제)

**비용 분석**

| 항목 | 비용 | 비고 |
|------|------|------|
| 음이온 수지 (3kg) | $360 | 초기 투자 |
| 재생 시약 (NaOH) | $20 | 재생당 |
| 폐기 비용 | $50 | 폐기 시 |
| **배치당 비용** | **$370-430** | 재생/폐기 선택 |

**Route별 영향도**

```
Route A: 정제 D-Galactose (기본)
├─ Step 2.5 스킵 가능 (염분 미량)
├─ Step 6: 양이온만 사용 → $500/배치
└─ 총 후처리 비용: $500

Route B: 홍조류 중화 용액 (시나리오 1) + NaOH ★ 권장
├─ Step 2.5 추가: 음이온 수지 교환 (필수) → $370/배치
│  └─ SO₄²⁻, 레불린산, 포름산 등 부산물 제거
├─ Step 6: 양이온만 사용 → $500/배치
├─ 음이온 수지는 Step 6에서 불필요 (Step 2.5에서 제거)
└─ 총 후처리 비용: $870
   └─ 장점: 부산물 미리 제거 → 깨끗한 반응액
```

**Step 2.5 선택 기준**

```
✅ 추가 필수:
├─ 시나리오 1 (홍조류 원물) 사용 시 필수
├─ 음이온 부산물 대량 함유:
│  ├─ SO₄²⁻
│  ├─ 레불린산 (13%)
│  └─ 포름산 (5%)
├─ 깨끗한 반응액 필요 → 촉매 효율 향상
└─ NaOH 중화제 사용 시

❌ 스킵 가능:
├─ 기본 시나리오 (정제 D-Galactose)
└─ 염분 미량 → 음이온 부산물 없음
```

### Step 3: 전세포촉매 전환 (Whole Cell Biocatalysis)

**목적**
- D-Galactose를 D-Tagatose로 효소 전환
- 에피머화 반응 진행

**반응 조건**
- 촉매: 전세포 균주 (예: E. coli, L. lactis 등)
- 반응 온도: 30-37°C
- 반응 시간: 12-48시간 (종 및 세포농도에 따라 변동)
- 세포 농도: 10-20 g/L DCW
- 기질 농도: 100-300 g/L D-Galactose

**주요 반응식**
```
D-Galactose → D-Tagatose (L-ribulose isomerase 촉매)
```

**산출물**
- D-Tagatose 함유 용액 (전환율: 80-98%)
- 생균 세포 (다음 단계에서 제거)
- 미전환 D-Galactose

---

### Step 4: 세포 제거 (Cell Removal)

**목적**
- 반응 종료 후 사용된 전세포 촉매 제거
- 용액 투명도 향상

**방법 (선택)**
- **원심분리** (Centrifugation): 5,000-10,000 rpm, 10-15분
- **마이크로필트레이션** (Microfiltration): 0.2-1 μm
- **초과 여과** (Ultrafiltration): 분자량 선택적 제거

**산출물**
- 세포 제거된 D-Tagatose 용액
- 세포 침전물 (폐기 또는 재활용)

---

### Step 5: 탈색 (Decolorization)

**목적**
- 갈색 색소 및 불순물 제거
- 최종 산물의 백색도 향상

**탈색제 (선택)**
- **활성탄** (Activated Carbon): 2-10 g/L, 30-60분, 60-80°C
- **이온교환 수지** (Ion Exchange Resin): 특정 색소 흡착
- **세포배양 여액 침전** (Enzymatic clarification)

**처리 조건**
- 온도: 상온 또는 50-70°C
- 시간: 30-120분
- 여과: 0.22 μm 필터로 활성탄 제거

**산출물**
- 투명한 무색 D-Tagatose 용액
- 색소 흡착 고형물 (폐기)

---

### Step 6: 탈염 (Desalination)

**목적**
- 황산염 (SO₄²⁻) 제거
- 나트륨/칼슘 양이온 (Na⁺, Ca²⁺) 제거
- 용액 순도 향상 및 최종 산물 품질 개선

**필요한 이온 제거 대상 분석**

#### 탈염 필요 이온들
```
양이온 (Cations) - 반드시 제거 필요:
├─ Na⁺ (나트륨): Step 2에서 중화제 NaOH 사용 시 대량 함유
│  예: NaOH 56 kg/mol × 0.3 mol = Na⁺ 6-7 kg/배치
├─ K⁺ (칼륨): 미량 (홍조류 원물 함유 또는 중화제)
└─ Ca²⁺ (칼슘): Step 2에서 중화제 CaOH2 사용 시에만 발생

음이온 (Anions) - 반드시 제거 필요:
├─ SO₄²⁻ (황산염): Step 1에서 H₂SO₄ 사용, 완전 중화 불가능
│  예: H₂SO₄ 98 kg/mol × 0.2-0.3 mol = SO₄²- 3-5 kg/배치
├─ Cl⁻ (염화물): 미량 (환경 또는 부재료)
└─ HSO₄⁻ (과잉 황산 이온): 중화 불충분 시

이온 농도 추정 (중화 후 D-Galactose 용액):
├─ D-Galactose: ~110 g/L (당 분자, 전기 중성)
├─ Na+: ~55-70 mEq/L (약 1.3-1.6 g/L)
├─ SO4²-: ~50-100 mEq/L (약 2.4-4.8 g/L)
└─ Total Dissolved Solids: ~120-125 g/L
```

#### 양이온 & 음이온 교환기 필요 여부
```
★ 양이온 교환기 (Cation Exchanger): 필수 (MUST)
  - Na⁺ 제거 필수 (최종 산물 순도)
  - 탈염 후에도 Na⁺가 남으면 제품 백색도 저하
  - Resin: Amberlite IRC120 (H⁺ form, Strong Acid Cation)
  - 용도: Na⁺/K⁺ → H⁺ 교환

★ 음이온 교환기 (Anion Exchanger): 강력 권장 (STRONGLY RECOMMENDED)
  - SO₄²⁻ 제거 (전기 전도도 감소)
  - 지만 양이온 교환만으로도 기술적으로 가능한 경우:
    ├─ 양이온만 사용 후: Na⁺ → H⁺로 치환 → pH 저하 (산성화)
    ├─ 결과: H₂SO₄ 용액 상태
    └─ SO₄²-는 여전히 남음 (conductivity 높음)

  - 음이온 교환 적용 시:
    ├─ SO₄²⁻ → OH⁻ 교환
    ├─ H⁺ + OH⁻ → H₂O 생성
    └─ 결과: 순수한 D-Galactose 수용액

결론:
양이온만 사용: 가능하지만 SO₄²- 함유 → 최종 제품 순도 ↓
양이온 + 음이온: 권장 (고순도 달성)

선택 기준:
├─ 고순도 필요 (>99%): 양이온 + 음이온 동시 사용
├─ 중순도 가능 (>95%): 양이온만 사용 (비용 절감)
└─ Scenario 1에서: 최종 순도 기준에 따라 결정
```

**방법 (추천 순서)**

#### 6.1 이온교환 (Ion Exchange)
```
Option A: 양이온만 사용 (비용 절감) ★ 기본 시나리오 추천
- Cation Exchanger (양이온): Na⁺ 제거
  Type: Strong Acid Cation (SAC)
  Resin: Amberlite IRC120, Dowex 50X
  회복력: Na⁺ → H⁺ (결과: 약산성 용액)
  수지 비용: $100/kg × 5 kg = $500/배치
  교체 주기: 12-18개월 (정제 피드, 염분 미량)

Option B: 양이온 + 음이온 (고순도 권장) ★ 시나리오 1 (홍조류)
- Cation Exchanger (양이온): Na⁺, K⁺, Ca²⁺ 제거
  Resin: Amberlite IRC120 (H⁺ form)
  수지 비용: $100/kg × 5 kg = $500/배치

- Anion Exchanger (음이온): SO₄²⁻, Cl⁻ 제거
  Type: Strong Base Anion (SBA)
  Resin: IRA-400, Amberlite IRA93 (OH⁻ form)
  수지 비용: $120/kg × 3 kg = $360/배치

- 순서: Cation (H⁺화) → Anion (OH⁻화) → 최종 정제
- 교체 주기: 6-9개월 (중화 용액, SO₄²⁻ 대량)
```

**처리 조건**
- 용액 흐름: 2-5 bed volume/시간
- 온도: 상온 (20-25°C)
- 전환: Cation → Anion → Cation 순환 권장

#### 6.2 대체 방법 (선택)
- **한외여과** (Ultrafiltration): 분자량 선택적 제거
- **역삼투** (Reverse Osmosis): 고염도 제거 (고비용)
- **전기투석** (Electrodialysis): 에너지 효율적

**산출물**
- 고순도 D-Tagatose 용액 (염 농도 <500 ppm)
- 포화 이온교환 수지

---

### Step 7: 건조 (Drying)

**목적**
- 용액에서 고체 D-Tagatose 결정 회수
- 보관 및 운반 용이화

**건조 방법 (선택)**

#### 7.1 분무건조 (Spray Drying)
- 온도: 80-120°C
- 장점: 빠른 속도, 결정 크기 제어 가능
- 단점: 초기 투자 비용 높음

#### 7.2 동결건조 (Freeze Drying)
- 온도: -20~-40°C, 진공 48-72시간
- 장점: 열에 민감한 화합물 보호
- 단점: 긴 처리 시간, 고비용

#### 7.3 직접 건조 (Direct Drying)
- 온도: 50-80°C
- 진공: 0.5-1 atm
- 시간: 12-30시간
- 장점: 비용 효율적, 결정도 높음

**산출물**
- 고순도 D-Tagatose 고체 (순도 >98%)
- 함수율: <5%

## Key Considerations

### 공정 설계 시 주요 사항

**1. 중화제 선택 (NaOH vs CaOH₂) - 핵심 전략**

#### 경제성 기준 선택

**NaOH 루트 (권장: 기본 시나리오 + 대규모)**
```
장점:
├─ 빠른 중화 (안정성 우선)
├─ 정제 피드 사용 시 음이온 수지 불필요 (비용 절감)
├─ 대규모(≥1000L)에서 단위비용 최저

단점:
├─ 중화된 홍조류(미정제) 사용 시 음이온 수지 필수 (비용 증가)
└─ SO₄²⁻ 축적으로 수지 교체 주기 단축 (2-3개월)

비용 분석:
├─ 기본 시나리오: 양이온만 사용 → $500/배치 (최소)
└─ 시나리오 1: 양+음이온 → $860/배치 (음이온 $360)
   ├─ 음이온 교체 주기: 2-3개월
   └─ 연간 음이온 비용: $360 × 6-9회 = $2,160-2,880

선택 시기:
├─ ✓ 기본: 정제 D-Galactose (음이온 불필요)
├─ ✓ 대규모: ≥1000L (생산량 많아서 수지 비용 상대적 저감)
└─ ✓ 신속 반응: 배치 주기 단축 필요할 때
```

**CaOH₂ 루트 (검토 중단) 📌 기록만 유지**
```
검토 결과 중단 사유:
├─ 음이온 수지의 필요성: SO₄²⁻뿐만 아니라
│  └─ 레불린산, 포름산 등 유기산 부산물도 제거 필수
├─ CaSO₄ 침전 처리 비용 (+$100/배치) 발생
├─ CaOH₂ 중화 속도 느림 (생산 효율 저하)
└─ 경로 B (NaOH)가 기술적/경제적으로 우수

결론:
└─ NaOH 중화 + Step 2.5 음이온 수지 → 최우수 경로
```

#### 핵심 비용 변수 (심화)

```
1. 음이온 수지 단가 ($120/kg)
   ├─ 현재 기준: Amberlite IRA-400, IRA-93 (SBA)
   ├─ 배치당 충전량: 3kg (표준)
   ├─ 배치비용: $360
   └─ 주요 절감안: 수입 제품 (인도/중국) → $80-100/kg (미검증)

2. 수지 교체 주기 (생명시간)
   ├─ NaOH 루트: 2-3개월 (SO₄²⁻ 축적)
   │  ├─ 연간 6-9회 교체 필요
   │  └─ 연간 비용: $360 × 6-9 = $2,160-2,880
   │
   ├─ CaOH₂ 루트: 12-18개월 (SO₄²⁻ 사전 제거)
   │  ├─ 연간 1-2회 교체 필요
   │  └─ 연간 비용: $360 × 1-2 = $360-720
   │
   └─ 절감액: $1,440-2,520/년 (CaOH₂ 우수)

3. 스케일 효과 (처리량 vs 고정비)
   ├─ 음이온 수지는 "배치당" 고정비
   │  ├─ 100L 배치: 3kg 수지 필요 → $360/배치
   │  ├─ 1000L 배치: 25kg 수지 필요 → $3,000/배치 (8배)
   │  └─ 단위 비용: $3.60/L → $3.00/L (약간 저감)
   │
   ├─ 교체 주기는 "처리량 누적" 기준
   │  ├─ 작은 배치(100L): 주 2회 → 월 8회
   │  ├─ 큰 배치(1000L): 주 1회 → 월 4회
   │  └─ 대규모일수록 상대적으로 덜 교체 필요
   │
   └─ 결론: 100-500L에서 CaOH₂ 유리, ≥1000L에서 NaOH 유리

4. 기술적 제약
   ├─ NaOH: 제약 거의 없음 (표준 공정)
   └─ CaOH₂:
      ├─ CaSO₄ 침전 제거 필수 (추가 여과 단계)
      ├─ Ca²⁺ 양이온 제거 필수
      └─ 반응 불완전 가능성 (수소 이온 부족)
```

#### 현재 권장사항 (정리)

```
★ 기본 시나리오 (정제 D-Galactose): 현재 추진
   └─ NaOH 사용 (확정)
      ├─ 정제 피드 = 음이온 부산물 없음
      ├─ Step 2.5 스킵 (비용 절감)
      ├─ Step 6: 양이온만 충분 ($500/배치)
      └─ 최저 비용, 최고 생산량

★ 시나리오 1 (홍조류 원물, 향후 검토):
   └─ NaOH + Step 2.5 음이온 수지 (필수)
      ├─ SO₄²⁻, 레불린산, 포름산 제거
      ├─ Step 2.5: $370/배치 (필수 공정)
      ├─ Step 6: 양이온만 $500/배치
      └─ 깨끗한 반응액 → 촉매 효율 향상

   CaOH₂ 루트는 폐기 (경제성 및 기술적 이유):
   └─ 음이온 부산물이 여전히 제거 필수
   └─ CaSO₄ 침전 처리 비용 (+$100)
   └─ 경로 B (NaOH)가 모든 면에서 우수
```

**2. 전환 효율**
- E. coli 대비 L. lactis: 더 높은 안전성 (GRAS)
- 전환율 목표: 90-98% (조건 최적화 필요)
- 미전환 D-Galactose는 후단에서 재활용 가능

**3. 탈염 전략**
- Cation + Anion 이온교환 조합 사용
- 수지 재생: 묽은 산/염기 사용으로 경제성 확보
- 다중 단계: 처음엔 대용량 처리, 마지막엔 폴리싱 (Polishing) 용

**4. 건조 방법**
- 직접 건조 (50-80°C): 비용 효율적, 가장 실용적
- 결정화 단계 추가 가능: 수용성 용액 냉각으로 D-Tagatose 결정 회수

**5. 폐기물 관리**
- 탈염 포화 수지: 재생 또는 폐기 처리
- 세포 침전물: 바이오매스 또는 사료로 재활용 검토
- 여과액: 환경 기준 충족 후 처리

**6. 품질 관리**
- 최종 산물: HPLC 분석으로 순도 확인 (>98% 목표)
- 중간 산물: pH, 색상도, 염도 모니터링
- 함수율 측정: Karl-Fischer 적정법

**7. 음이온 교환 수지 공정 상세 (Step 6 심화)**

#### 음이온 수지의 역할 및 필요성
```
Step 1에서 황산(H₂SO₄) 사용 → SO₄²⁻ 음이온 대량 생성
Step 2에서 중화 → H⁺ 중화 가능, 하지만 SO₄²⁻는 남음

문제점:
├─ 양이온만 제거 시: H⁺ → Na⁺로 교환, 하지만 SO₄²⁻는 용액에 남음
├─ 결과: 산성 황산염 용액 (pH 저하, 제품 순도 악화)
├─ 최종 제품: 염도 높음, 저장 안정성 ↓
└─ 음이온 수지 필수 → SO₄²⁻ 제거

음이온 수지 원리:
├─ SO₄²⁻ + Anion Resin(-OH) → Resin(-SO₄) + OH⁻
├─ H⁺ + OH⁻ → H₂O (중성화)
└─ 결과: 순수한 D-Galactose 수용액 (pH 6-7)
```

#### 중화제에 따른 음이온 수지 영향도
```
Route 1: NaOH 중화 경로
├─ Step 2 산출: D-Galactose + Na⁺ + SO₄²⁻
├─ Step 6.1 (양이온): Na⁺ → H⁺ (비용: $500/배치)
├─ Step 6.2 (음이온): SO₄²⁻ → OH⁻ (비용: $360/배치)
├─ 수지 교체 주기: 6-9개월 (SO₄²⁻ 축적)
└─ 연간 음이온 수지 비용: $360 × 6-9회 = $2,160-2,880/년

Route 2: CaOH₂ 중화 경로
├─ Step 2 산출: D-Galactose + Ca²⁺ + CaSO₄↓ (침전)
├─ CaSO₄ 처리: 여과 제거 (비용: $100/배치)
├─ Step 6.1 (양이온): Ca²⁺ → H⁺ (비용: $500/배치)
├─ Step 6.2 (음이온): "최소" SO₄²⁻만 제거 (비용: 저감)
├─ 수지 교체 주기: 12-18개월 (SO₄²⁻ 적음)
└─ 연간 음이온 수지 비용: $360 × 2-3회 = $720-1,080/년
```

## 🔄 **시나리오별 경제분석 (Scenario-based TEA Comparison)**

### **코톤(Cotton) 산분해물 조성 (Acid Hydrolysate Composition)**

**실제 코톤 산분해물의 질량 기준 성분:**
```
D-Galactose (갈락토즈):     70.0%  ★ 주요 목표 산물
Glucose (글루코즈):         11.5%
Levulinic acid (레불린산):  13.0%
Formate (포름산염):          5.0%
기타 (Others):               0.5%
───────────────────────────────
Total:                      100.0%
```

**시사점:**
- D-Galactose 함유율 70% → 홍조류 원물 추출 시 기준값 (현재 50% 가정에서 상향 조정 필요)
- Formate 5% 함유 → 내재 Formate (추가 Sodium Formate 필요량 감소 가능)
- 레불린산 13% → 가수분해 부산물 (Step 1-2에서 제거/분리 필요)
- 글루코즈 11.5% → 다당류 분해 부산물 (Tagatose 선택성 영향)
- 정제 효율: 70% D-Galactose 중 최종 회수율 약 60-70% (부산물 제거 후)

### **시나리오 정의 & 용어 정리**

#### **기본 시나리오 (Default): 정제 D-Galactose 구매**
- **피드**: 정제 D-Galactose (식품급, 99% 순도) 구매
- **가격**: $2.00/kg (산업용 정제 제품)
- **특징**: 고순도, 염분 미량, 즉시 반응 가능
- **공정**: Step 3-7 (바이오촉매 + 정제/건조)

#### **시나리오 1: 홍조류 원물 통합 (중화된 용액)**
- **피드**: 홍조류 건조 바이오매스 → 중화된 용액 (미정제)

- **특징**: 정제 단계 생략, 중화된 용액(Na⁺/SO₄²⁻ 함유)을 바로 반응

#### 가정 조건
```
코톤(Cotton) 산분해물 (Algal Biomass - Cotton Acid Hydrolysate):

├─ D-Galactose 함유: 70% (w/w, 실제 코톤 산분해물 조성)

├─ Step 1 수율 (가수분해): 85% (코톤 → D-Galactose 함유 산성 용액 + 부산물)

   └─ Step 6 탈염에서 제거 필요

순계 수율 (Step 1-2): 85% × 92% = 78.2%

필요한 홍조류:
110 kg D-Galactose (용액 기준) ÷ 0.782 = 141 kg 홍조류 필요

배치당 비용 (Step 1-2):
├─ 홍조류: 141 kg @ $0.75/kg = $106
├─ 가수분해 (H₂SO₄, 에너지): $0.40/kg × 141 = $56
├─ 중화 (NaOH), 여과: $0.25/kg × 141 = $35
└─ Step 1-2 합계: $197 (vs 정제 D-Gal 구매 $220)

결과물: **중화된 D-Galactose 용액** (Step 3 투입 전)
├─ 성분: ~110 kg D-Galactose + 염분 (Na⁺ ~5-7 kg, SO₄²⁻ ~3-5 kg)
├─ 상태: pH 6-7 (중화 완료), 용액 상태

```

### **결론 및 권장사항**

#### ✅ **기본 시나리오 (정제 D-Galactose 구매): 최적**
```
장점:

└─ 피드 비용 안정: 정제 제품 시장 확립

├─ 최우선 선택지
└─ 기술 검증/최적화 집중
```

#### ❌ **시나리오 1 (홍조류 원물 통합): 비경제적**
```
문제점:
├─ "중화된" ≠ "정제된" (염분 대량: Na⁺, SO₄²⁻)
├─ Step 6 탈염에서 제품 손실 증가 (생산량 -17.9%)
├─ 후처리 비용 증가 (양+음이온 필수, 수지 비용 2배)
├─ 단위 비용 악화: $30.69 → $36.89/kg (+20.2%)
├─ 연간 생산 감소: 34,375 → 28,219 kg
└─ 경제성 악화

핵심 이슈:
├─ 피드 비용 절감 ($220 → $197) < 생산량 손실 (-17.9%)
└─ 정제 단계 없으면 탈염 효율 저하

향후 검토 조건:
└─ Step 1-2에서 "정제 공정" 추가 후 경제성 재평가 필요

```
